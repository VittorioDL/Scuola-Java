NOME DEL PROGETTO: GUIclient
OBIETTIVO DEL PROGETTO:

    Client grafico:

    Il client dovrà visualizzare (dopo aver effettuato in modo positivo la connessione) in un'unica finestra:

    1.un riquadro con i nickname degli utenti collegati (aggiornato almeno ogni 5 secondi);
    2.un riquadro con i messaggi per l'utente (aggiornato almeno ogni 2 secondi);
    3.un ComboBox per la selezione dell'utente a cui inviare un messaggio (se non si seleziona il messaggio è per tutti);
    4.un campo di testo per l'inserimento del messaggio;
    5.un pulsante per la disconnessione.
